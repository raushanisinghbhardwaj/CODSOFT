## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/raushanisinghbhardwaj/CODSOFT.git
   ```

2. Open the project in Visual Studio Code.

3. Compile the program:
   ```bash
   g++ filename.cpp -o output
   ```

4. Run the executable:
   ```bash
   ./output
   ```
